# Field Reference

## Required Fields

### Word
The main vocabulary word to learn.

**Example**: `Audience`

---

### Meaning
Definition or translation of the word.

**Example**: `a group of people who watch or listen to something`

---

## Optional Fields

### IPA
Phonetic transcription in IPA format.

**Example**: `ˈɔːdiəns`

> Note: Don't include slashes, they're added automatically

---

### PartOfSpeech
The grammatical category. Auto-normalizes to standard format.

| Input | Display |
|-------|---------|
| `noun`, `n`, `n.` | Noun |
| `verb`, `v`, `v.` | Verb |
| `adjective`, `adj`, `adj.` | Adj |
| `adverb`, `adv`, `adv.` | Adv |
| `n [c]`, `countable` | Noun [C] |
| `n [u]`, `uncountable` | Noun [U] |

---

### ExampleSentence
Example sentence using the word.

**Example**: `The audience applauded loudly.`

> The word is automatically highlighted in the example

---

### collocations pattern
Common word combinations.

**Example**: `large audience, target audience, captive audience`

---

### image
Image field for visual learning.

**Usage**: Add image in Anki editor (paste or drag)

---

### Notes
Additional notes, mnemonics, or tips.

**Example**: `Remember: audience = people who HEAR (audio-)`

---

## Auto Fields (Anki Built-in)

### Tags
Anki's built-in tags field. Space-separated tags are automatically converted to badges.

**Example**: `vocabulary english B2` → displays as 3 separate badges
